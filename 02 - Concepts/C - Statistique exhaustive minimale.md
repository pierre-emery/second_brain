Une [[C - Statistique Exhaustive]] $T(X)$ est dite minimale si pour totue autre statistique exhaustive $T^*(X)$,$T(X)$ est une fonction de $T^*(X)$.
Autrement dit, si $T^*(X)=T^*(Y)$ alors $T(X)=T(Y)$.

# Attention
il est difficile de trouver une statistique exhaustive minimale à partir de cette définition, il vaut mieux utiliser [[T - Statistique exhaustive minimale]]

## Unicité
Les statistiques exhaustives minimales ne sont pas uniques, par contre elles sont essentiellement uniques. 
Comme auparavant, une [[C - Fonction Injective]] d’une statistique exhaustive minimale sera également une statistique exhaustive minimale. 
Notons que la fonction de [[C - Vraisemblance]] est une statistique exhaustive minimale.
