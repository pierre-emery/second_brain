Un SGBD est un logiciel qui permet de :
## Définir la base
- types de données
- structure
- contraintes
## Manipuler la base
- requêtes
- insertion
- suppression
- mise à jour
## Administrer la base
- gestion des accès concurrents
- gestion des droits utilisateurs
- stockage physique des données
## Historique des SGBD
#### 1ère génération (années 70)
- Modèles hiérarchiques et réseaux
- Navigationnels
#### 2e génération (années 80)
- Modèle relationnel
- SQL
- Oracle, DB2, SQL Server
#### 3e génération (années 90)
- Extension du relationnel
- Objets, multimédia
#### Évolution actuelle
- Internet
- Données non structurées
- Entrepôts de données
- Data mining
