## Définition
Une base de données est un ensemble de données :
- fortement structurées
- persistantes
- définies par un schéma
- gérées par un SGBD

## Propriétés
- Structurée
- Cohérente
- Porteuse de sens (pas des données arbitraires)
- Partagée entre plusieurs utilisateurs
- Définie pour un besoin précis ([[C - Univers du Discours]])

# Acteurs d'une BD

#### Concepteur
- Conçoit le schéma de la base
#### Administrateur (DBA)
- Gère la sécurité, l’intégrité, les performances
#### Utilisateur final
- Consulte ou modifie les données

Une même personne peut jouer plusieurs rôles.
# Avantages des BD
- Réduction de la redondance
- Évite les incohérences
- Partage des données entre applications
- Sécurité (droits d’accès)
- Intégrité des données (contraintes)
- Indépendance entre données et applications
- Optimisation des accès selon les usages
