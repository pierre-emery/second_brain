L'organisation des données dans un tableau d'observations par attributs est le plus pratique/standard en analyse des données.

Il est important de reconnaître les types de valeurs que chaque attribut prend afin de comprendre quelles opérations peuvent y être appliquées
Les valeurs d'attributs peuvent être divisées en deux types [[C - Attributs Qualitatifs]] et [[C - Attributs Quantitatif]] qui peuvent être à nouveau divisé en deux.

Les types d'attributs peuvent être considérés via les opérations qui peuvent leur être appliquées:
- Comparaison (= et $\neq$ ) c'est bon pour tous les types
- Ordre (> et <) c'est bon pour tous les types saufs les [[C - Attributs Nominaux]]
- + et - seulement pour [[C - Attributs Quantitatif]]
- / et x seulement pour [[C - Attributs de Ratios]]
D'autres opérations comme [[C - Moyenne]], [[C - Médiane]] ou la [[C - Corrélation]] peuvent être inapplicables pour certains types mais applicables à d'autres.
