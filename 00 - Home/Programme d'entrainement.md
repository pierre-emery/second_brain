
## Semaine (climbing : lun/mer/sam) (training : mar/jeu/ven) (rest : dim)

### Lundi — CLIMB : **Projets / Limit (Kilter 40°)** (75–110 min)

- Échauffement 20–30 min
- **4 à 6 problèmes max**
    - 2 très durs : 3–5 essais
    - 2–4 durs mais faisables : 2–4 essais
    - repos 3–5 min
- Stop quand la qualité chute (peau, doigts, mouvement)
**Soir : hangboard** → **5 min**  
**Soir : mobilité** → 10–20 min

---
### Mardi — TRAINING : **Jambes + Core (focus front lever)** (60–90 min)
**A. Jambes**
- Split squats or Pistol Squats(veste) : 4×6–10 / jambe
- Step-ups : 3×8–12 / jambe
- Hip thrust ou RDL improvisé : 4×8–12
- Hamstring curls anneaux : 3×6–12

**B. Core (front lever)**
- Tuck front lever hold : 5–8×8–12s (repos 60–90s)
- Front lever raises (tuck) _ou_ negatives contrôlées : 4×3–6
- Hollow hold : 3×20–40s

**Soir : hangboard** → **10 min easy**  
**Soir : mobilité** → 15–25 min (hanches/ischios)

---
### Mercredi — CLIMB : **Volume / flash / lecture** (75–105 min)
- V4–V5 : flash / 2 essais max
-  10 min mouvement facile (traversée / “flow”) or 4x4

**Soir : hangboard** → **10 min easy**  
**Soir : mobilité** → 15–20 min

---
### Jeudi — TRAINING : **Block lifts + Weighted pull-ups + Rows + Scap + Weighted dips** (60–90 min)

**A. Block lifts (priorité doigts)**
- Edge 20 mm : 5–8×3–5 lifts (repos 2–3 min)
- Wide pinch : 4–6×8–12s (facile/modéré)
**B. Force upper**
- Tractions lestées : 4×2–4 
- Rows anneaux : 3–4×8–12
- Scap pull-ups : 3×8–12
- Weighted dips : 4×6–10

**Soir : hangboard** → **5 min ultra easy** 
**Soir : mobilité** → 10–20 min (épaules/poignets)

---
### Vendredi — TRAINING : **Curls → Pull-ups (superset) + Lock-offs + Pike push-ups + Handstand** (60–90 min)

**A. Handstand  10–20 min
- Wall holds 6–10×20–40s
- shrugs / taps si stable  
**B. Pike push-ups**
- 4×6–12
**C. Superset curls → pull-ups** (modéré)
- Bicep curls : 8–12 reps
- Pull-ups strict : 4–8 reps
- 3–4 rounds, **2 reps en réserve** (pas à l’échec)

**D. Lock-offs (angles)**
- 3 angles (≈120°, 90°, 45°) : 3 séries de 5–10s / angle  
    Repos 60–90s, **qualité**.

**Soir : hangboard** → **10 min easy** 
**Soir : mobilité** → 15–25 min

---
### Samedi — CLIMB : **Projets (gym / Kilter)** (75–120 min)
- Projets + repos longs
- Termine par 10 min très facile (dérouler / récupérer)

**Soir : hangboard** → **5 min ultra easy** (ou skip si peau/doigts cuits)  
**Soir : mobilité** → 10–20 min

---
### Dimanche — REST
**Hangboard** → **10 min easy** (si zéro douleur)  
**Mobilité** → 15–25 min (longue séance jambes/ischios)

---

## Protocole “easy hangboard tous les jours”

**Standard (10 min)** : 10×(10s on / 50s off) sur 20 mm open-hand, avec assistance pieds, RPE 2–3/10.  
**Ultra-easy (5 min)** : 5×(10s / 50s) mêmes règles.

 10 min : mar / mer / ven / dim  
 5 min : lun / jeu / sam
