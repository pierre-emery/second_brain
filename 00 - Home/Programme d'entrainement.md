## Semaine (structure fixe)

### Lundi — Kilter 40° : **Limit / Force spécifique doigts** (75–110 min)
Objectif : project climbing
- Échauffement 20–30 min (progressif)
- **Bloc principal : 4 à 6 problèmes max**
    - 2 “limit” (vraiment durs) → **3–5 essais** chacun
    - 2–4 “hard but doable” → **2–4 essais** chacun
    - Repos **3–5 min** entre essais sérieux
- Stop quand la qualité baisse (peau/douleurs/technique qui s’écroule)

**Soir : hangboard super easy** 
**Soir : mobilité**  10–20 min (hanches/ischios + épaules léger)

---
### Mardi — Training : **Lower + Core** (60–90 min)
- Split squats (veste 22 kg) : 4×6–10 / jambe
- Step-ups (veste) : 3×8–12 / jambe
- Hip thrust / RDL improvisé : 4×8–12
- Hamstring curls aux anneaux : 3×6–12
- Core : hanging knee raises 4×8–12 + side plank 3×30–45s
**Soir : hangboard**
**Soir : mobilité** 15–25 min (jambes + handstands)

---
### Mercredi — Grimpe : **Volume / flash / lecture** (75–105 min)
Séance “V4–V5 je grimpe beaucoup”.
- Majorité : flash / 2 essais max
- Bonus 10 min : traversée facile / mouvement continu (en endurance)
**Soir : hangboard easy** 10 min  
**Soir : mobilité** 15–20 min (mix jambes + épaules)

---

### Jeudi — Grimpe : **Avant-bras (progressif, faisable)** (75–105 min)

Ici on remplace le “4 minutes on/off” par une progression réaliste.

**Principe :** tu choisis un terrain **facile** (idéalement Kilter **plus facile** : grosses prises / mouvements simples ; si possible baisse l’angle, sinon baisse beaucoup la difficulté).  
Tu dois **finir toutes les reps**.

**Bloc principal (au choix) :**

- Semaine 1–2 : **12 × (30s on / 30s off)**
    
- Semaine 3–4 : **10 × (45s on / 45s off)**
    
- Semaine 5–6 : **8 × (60s on / 60s off)**
    
- Semaine 7–8 : **6 × (90s on / 90s off)**


→ Tu vises un burn avant-bras ~**6–7/10**, mais **sans exploser**.

**Soir : hangboard easy** 
**Soir : mobilité** 10–20 min (plutôt épaules/poignets)

---
### Vendredi — Training : **Finger Strength (1 vraie séance) + Upper “maintenance”** (60–90 min)
Comme tes doigts sont le facteur limitant, on met **1 séance “force doigts”** par semaine (en plus du lundi limit).
**A) Hangboard force (avant le reste, quand tu es frais)**  
**Max hangs (20 mm, open-hand)**
- 5–6 × 10s, repos 2:30–3:00
- Intensité : RPE **8.5–9/10** (1–2 secondes “en réserve”, pas à l’échec)
- Ajuste avec **assistance pieds** ou charge légère (progression très lente)

**B) Upper (maintenance + antagonistes)**
- Tractions lestées : 4×2–4 
- Rows anneaux : 3–4×8–12
- Scap / épaules : scap pull-ups 3×8–12 + Y-T-W 2×8–12
- Poussée (pompes lestées ou dips) : 4×6–12

**Soir : hangboard easy** 
**Soir : mobilité**  15–25 min

---
### Samedi — Repos

**Soir : hangboard easy** 
**Soir : mobilité** 

---
### Dimanche — Training : **Upper B + Handstand + Core (contrôle)** (60–90 min)
- Handstand (10–20 min) : wall holds 6–10×20–40s + shrugs/taps si stable  
- Pike push-ups : 4×6–12
- Ring support hold : 5×15–30s
- Core : L-sit tuck 6×10–20s + dead bug 3×10/côté

**Soir : hangboard easy** 
**Soir : mobilité**

## Routine mobilité (15–25 min, simple)

**Jambes/hanches (10–15 min)**
- 90/90 : 2×60–90s
- Couch stretch : 2×60s/côté
- Ischios (strap/pike) : 2×60–90s
- Frog stretch : 2×60s

**Épaules/poignets (5–10 min)**
- Wall slides : 2×10–15
- Thoracic opener : 2×60s
- Wrist flex/ext : 2×45–60s
## Semaine (climbing : lun/mer/sam) (training : mar/jeu/ven) (rest : dim)

### Lundi — CLIMB : **Projets / Limit (Kilter 40°)** (75–110 min)

- Échauffement 20–30 min
- **4 à 6 problèmes max**
    - 2 très durs : 3–5 essais
    - 2–4 durs mais faisables : 2–4 essais
    - repos 3–5 min
- Stop quand la qualité chute (peau, doigts, mouvement)
**Soir : hangboard** → **5 min**  
**Soir : mobilité** → 10–20 min

---
### Mardi — TRAINING : **Jambes + Core (focus front lever)** (60–90 min)
**A. Jambes**
- Split squats or Pistol Squats(veste) : 4×6–10 / jambe
- Step-ups : 3×8–12 / jambe
- Hip thrust ou RDL improvisé : 4×8–12
- Hamstring curls anneaux : 3×6–12

**B. Core (front lever)**
- Tuck front lever hold : 5–8×8–12s (repos 60–90s)
- Front lever raises (tuck) _ou_ negatives contrôlées : 4×3–6
- Hollow hold : 3×20–40s

**Soir : hangboard** → **10 min easy**  
**Soir : mobilité** → 15–25 min (hanches/ischios)

---
### Mercredi — CLIMB : **Volume / flash / lecture** (75–105 min)
- V4–V5 : flash / 2 essais max
-  10 min mouvement facile (traversée / “flow”) or 4x4

**Soir : hangboard** → **10 min easy**  
**Soir : mobilité** → 15–20 min

---
### Jeudi — TRAINING : **Block lifts + Weighted pull-ups + Rows + Scap + Weighted dips** (60–90 min)

**A. Block lifts (priorité doigts)**
- Edge 20 mm : 5–8×3–5 lifts (repos 2–3 min)
- Wide pinch : 4–6×8–12s (facile/modéré)
**B. Force upper**
- Tractions lestées : 4×2–4 
- Rows anneaux : 3–4×8–12
- Scap pull-ups : 3×8–12
- Weighted dips : 4×6–10

**Soir : hangboard** → **5 min ultra easy** 
**Soir : mobilité** → 10–20 min (épaules/poignets)

---
### Vendredi — TRAINING : **Curls → Pull-ups (superset) + Lock-offs + Pike push-ups + Handstand** (60–90 min)

**A. Handstand  10–20 min
- Wall holds 6–10×20–40s
- shrugs / taps si stable  
**B. Pike push-ups**
- 4×6–12
**C. Superset curls → pull-ups** (modéré)
- Bicep curls : 8–12 reps
- Pull-ups strict : 4–8 reps
- 3–4 rounds, **2 reps en réserve** (pas à l’échec)

**D. Lock-offs (angles)**
- 3 angles (≈120°, 90°, 45°) : 3 séries de 5–10s / angle  
    Repos 60–90s, **qualité**.

**Soir : hangboard** → **10 min easy** 
**Soir : mobilité** → 15–25 min

---
### Samedi — CLIMB : **Projets (gym / Kilter)** (75–120 min)
- Projets + repos longs
- Termine par 10 min très facile (dérouler / récupérer)

**Soir : hangboard** → **5 min ultra easy** (ou skip si peau/doigts cuits)  
**Soir : mobilité** → 10–20 min

---
### Dimanche — REST
**Hangboard** → **10 min easy** (si zéro douleur)  
**Mobilité** → 15–25 min (longue séance jambes/ischios)

## Protocole “easy hangboard tous les jours”

**Standard (10 min)** : 10×(10s on / 50s off) sur 20 mm open-hand, avec assistance pieds, RPE 2–3/10.  
**Ultra-easy (5 min)** : 5×(10s / 50s) mêmes règles.

 10 min : mar / mer / ven / dim  
 5 min : lun / jeu / sam
