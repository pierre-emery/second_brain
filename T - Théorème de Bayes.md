La probabilité conditionnelle de la classe C sachant l'observation x peut être calculée comme $$P(C\mid x) = \frac{P(x\mid C)P(C)}{P(x)}$$ en utilisant la probabilité conditionnelle de x sachant C et les probabilités de x et de C.

Dans le cadre de [[A - Classification Naïve Bayésienne]]
- P(C) est la probabilité a priori de C ([[C - Distribution a priori]])
- P(x|C) est la [[C - Vraisemblance]] de C sachant x
- P(x) est la distribution d'observations 
- P(C|x) est la probabilité a posteriori de C sachant x ([[C - Distribution a posteriori]])
