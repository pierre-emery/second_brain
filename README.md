# Second Brain (Obsidian Vault)

## Structure (conventions)

- **MOC/**  
  Index (“Maps of Content”) : pages qui organisent les sujets et pointent vers les notes importantes.

- **Concepts/**  
  Notions et définitions (ex: probabilité, stats, algèbre linéaire, optimisation, etc.).

- **Algorithmes/**  
  Méthodes et algorithmes (ex: PCA, K-Means, régression, DBSCAN, etc.).

- **Théorèmes & Preuves/**  
  Théorèmes, propositions, preuves et résultats importants.

- **Distributions/**  
  Lois/probabilités (définitions, propriétés, moments, usages).

- **Graphes/**  
  Types de graphiques + comment les produire (librairies, exemples de code, bonnes pratiques).

- **Loss Functions/**  
  Fonctions de perte et objectifs (définitions, propriétés, quand les utiliser).

## Utilisation (Obsidian)

1. Installer Obsidian : https://obsidian.md  
2. Cloner le repo :

```bash
git clone https://github.com/pierre-emery/second_brain
cd second_brain
```
3. Obsidian → Open folder as vault → sélectionner le dossier second_brain.
